# Superstore Sales Analysis

SQL and Tableau analysis of Superstore sales data with interactive dashboards and business insights.

## Tools Used
- SQL (DBeaver)
- Tableau
- GitHub

## Dataset
Sample Superstore dataset containing sales, profit, region, and category information.

## Business Questions Answered

### Sales by Region
- West region generated the highest sales.
- South region generated the lowest sales.

### Profit by Category
- Technology category produced the highest profit.
- Furniture category had significantly lower profit.

### Profit Margin by Category
- Technology and Office Supplies achieved ~17% profit margins.
- Furniture margin was only ~2.5%.

### Top States Analysis
- California and New York were among the most profitable states.
